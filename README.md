## 线上环境

<https://juejin-cms.daidr.me>

> **Note**
>
> 请注意，线上环境屏蔽管理面板（即所有修改在**开发阶段**均需要通过Git方式进行管理，保证线上环境和仓库托管代码是一致的）
>
> 包括**添加附件**、**增删改文章**、**修改接口定义**、**修改数据关系**

## 如何开发

### 1. 拉取代码

```bash
git clone https://github.com/campTeam/juejin-cms.git
```

### 2. 安装依赖

```bash
yarn
# 这里请务必使用 yarn 进行依赖管理，不要使用 npm
```

### 3. 启动项目

```bash
yarn develop
```
